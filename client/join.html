RULES:

This is a four player game, where each player has a hand of tiles.
The goal of the game is to have four sets of 3 tiles and one set of 2 tiles.
A 3 tiles set is defined as 3 of the same exact tile, or 3 consecutive tiles. A set of 2 tiles is defined as 2 exact same tiles.
Each player takes turns. On their turn a player will first gain a tile and then discard a tile to end their turn. The gained tile will either come from the previous player's discard or the draw pile. A player can only steal a tile from another player if it completes a set. The player will then have to reveal the completed set. A player can steal a tile out of turn if it completes a set of 3 identical tiles OR if it is the final tile needed to win. Otherwise, players can only steal in the order of their turn.

ADDITIONAL RULES:

A player can also win by having seven sets of 2, but this player can only obtain tiles by drawing from the pile. A player may have the additional opportunity to draw a tile when they obtain 4 of the same tile. When a player obtains it on their own, the exact tile is not revealed. A player can obtain it from another player in the same way a player can obtain the tile if their have 2 of the same tile. A set of four tiles acts as a set of 3.
